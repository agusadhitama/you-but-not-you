# 🕵🏽‍♂️ YOU BUT NOT YOPU 

> *"it doesn't just respond. it observes."*

---

## 🧠 about

**you but not you** is not a typical web project.

it's an interactive experiment
where a system begins to:

* mirror your thoughts
* track your behavior
* predict what you’re about to say

and eventually...

> it speaks before you do.

---

## 👁️ what happens inside?

this system quietly watches:

* how fast you type
* when you hesitate
* how often you repeat yourself

then it adapts.

not perfectly.
just enough to feel... uncomfortable.

---

## ⚙️ features

* real-time text mirroring
* simulated human typing delay
* behavior tracking system
* fake "AI learning" progression
* ghost typing takeover
* glitch & shadow text layers
* subtle psychological feedback
* ambient system interface

---

## 💀 experience

this project is designed to feel:

* slightly off
* subtly invasive
* strangely personal

you might think it's random.

it’s not.

---

## 🧪 tech

* HTML
* CSS (cinematic + glitch effects)
* Vanilla JavaScript (behavior simulation)

no frameworks.
no real AI.

just controlled illusion.

---

## 🎯 purpose

to explore:

* human-computer interaction
* illusion of intelligence
* psychological response to "being observed"
